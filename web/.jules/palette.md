## 2024-05-24 - [Skip-to-Content Link Addition]
**Learning:** Adding a visually hidden "skip-to-content" link directly inside the `<body>` element of the main `layout.tsx` is crucial for keyboard and screen reader users to bypass repetitive navigation blocks. The specific combination of Tailwind classes (`sr-only focus:not-sr-only focus:absolute focus:top-0 focus:left-0 focus:z-50 focus:p-4 focus:bg-emerald-900 focus:text-white`) provides an accessible, non-disruptive, yet highly visible focus state that matches the project's emerald color scheme.
**Action:** Always verify that layout components in Next.js App Router applications include this skip-to-content mechanism linked to the primary `<main>` anchor (`#main-content`) to satisfy WCAG bypass block requirements. Ensure the language of the label matches the document's `lang` attribute (e.g., German "Zum Hauptinhalt springen").
