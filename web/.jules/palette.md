## 2026-05-02 - Skip-to-Content Pattern
**Learning:** The application uses a specific Tailwind utility pattern (`sr-only focus:not-sr-only focus:absolute focus:top-0 focus:left-0 focus:z-50 focus:p-4 focus:bg-emerald-900 focus:text-white`) for visually hidden, keyboard-focusable elements like skip-to-content links. The primary container uses `id="main-content"`.
**Action:** Use this exact utility pattern and target ID for any future visually hidden accessibility links to maintain consistency with the app's design system.
