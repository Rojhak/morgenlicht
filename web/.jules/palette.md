## 2024-05-24 - Skip to Main Content Link Pattern
**Learning:** For this Next.js app using Tailwind CSS, a visually hidden skip-to-content link added directly inside the `<body>` of `app/layout.tsx` using the classes `sr-only focus:not-sr-only focus:absolute focus:z-50 focus:p-4 focus:bg-emerald-900 focus:text-white` provides a robust, reusable pattern for keyboard accessibility across all pages.
**Action:** Use this exact Tailwind utility pattern for any future visually hidden but focusable elements to ensure consistency with the app's emerald color scheme and z-index requirements.
