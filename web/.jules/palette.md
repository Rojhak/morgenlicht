## 2024-05-24 - [Skip-to-Content Link Placement in Next.js App Router]
**Learning:** In Next.js App Router applications, visually hidden accessibility links (like 'skip-to-content') should be placed directly inside the `<body>` tag of `layout.tsx` to guarantee correct indexing at the start of the document for keyboard navigation. Using Tailwind utility classes like `sr-only focus:not-sr-only focus:absolute focus:top-0 focus:left-0 focus:z-50 focus:p-4 focus:bg-emerald-900 focus:text-white` ensures proper positioning and focus visibility.
**Action:** Always verify that 'skip-to-content' links exist at the root layout and provide immediate focus visibility when tabbing.
